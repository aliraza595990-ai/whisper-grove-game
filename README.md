# 🌿 Whisper Grove Game

## جادوئی جنگل کی پرسکون دنیا

A complete **Cozy + Merge + Idle** mobile game built in HTML5/CSS3/JavaScript.

## 🎮 How to Play
- **Merge**: Click two identical items to combine them
- **Idle**: Items produce resources automatically
- **Collect**: Tap the coin button to collect idle earnings
- **Shop**: Buy new items to expand your grove

## 📱 Features
- ✅ 35 Unique Items across 5 Merge Chains
- ✅ Idle Resource Production
- ✅ Daily Reward System
- ✅ Quest & Achievement System
- ✅ Shop with 6 Items
- ✅ Offline Bonus (up to 8 hours)
- ✅ Auto-Save System
- ✅ Beautiful Animations & Sound Effects

## 🌳 Merge Chains
1. 🌱 **Plants** (10 levels) → Eternal Garden
2. 🦊 **Animals** (8 levels) → Luna Fox
3. 🏠 **Buildings** (6 levels) → Crystal Palace
4. 🪣 **Tools** (4 levels) → Magic Watering Can
5. 🍄 **Decorations** (7 levels) → Rainbow Gate
